| 1      | FREEZE/RESUME | Toggle pattern evolution | - | - |
| 2      | CAPTURE START | Mark capture start point | Quick-capture last 4 bars | - |
| 3      | CAPTURE STOP  | Mark capture end point | - | - |

| 8      | UNDO          | Undo last action | - | - |

12. Humanization
13. Velocity dynamics
14. Chaos factor
