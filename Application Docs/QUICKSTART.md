# Quick Start

## 1. Analyze E-GMD Collection

```bash
cd analysis
pip install -r requirements.txt
python analyzer.py
```

Output: `analysis/output/pattern_library.json`

## 2. Load M4L Device

- Open Ableton Live
- Drag `GrooveWanderer.amxd` to MIDI track
- Device auto-loads pattern library

## 3. Connect Launch Control

- MIDI In: Launch Control
- MIDI Out: Drum Rack

## 4. Start Wandering

- Patterns begin evolving immediately
- Adjust knobs to steer
- Hit buttons to capture/loop/export
